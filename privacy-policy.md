# 개인정보처리방침

**복약케어** 앱(이하 "앱")은 사용자의 개인정보를 중요시하며, 아래와 같이 개인정보처리방침을 안내합니다.

---

## 1. 수집하는 정보

본 앱은 다음과 같은 정보를 수집합니다:

- **약 정보**: 사용자가 직접 입력한 약 이름, 용량, 복용 시간
- **알람 설정 정보**: 복약 알림 시간 및 요일 설정

본 앱은 위 정보를 **사용자의 기기 내에만 저장**하며, 외부 서버로 전송하지 않습니다.

---

## 2. 정보 수집 목적

수집된 정보는 다음 목적으로만 사용됩니다:

- 복약 알림 기능 제공
- 약 복용 일정 관리
- 재부팅 후 알람 자동 복원

---

## 3. 정보 공유

본 앱은 사용자의 정보를 제3자와 공유하지 않습니다.

---

## 4. 사용 권한 안내

본 앱은 다음 권한을 사용합니다:

| 권한 | 사용 목적 |
|------|-----------|
| 알림 권한 | 복약 알림 표시 |
| 진동 권한 | 알람 진동 기능 |
| 정확한 알람 권한 | 정확한 시간에 알람 실행 |
| 부팅 완료 권한 | 재부팅 후 알람 자동 복원 |
| 화면 잠금 해제 권한 | 잠금화면에서 알람 표시 |

---

## 5. 데이터 보관 및 삭제

- 모든 데이터는 사용자 기기 내에만 저장됩니다
- 앱을 삭제하면 모든 데이터가 함께 삭제됩니다
